# Habit Tracker
A single-page habit tracker built with vanilla HTML, CSS, and JavaScript.

## Live Demo
https://farmanali108.github.io/Habit-Tracker/

## How to Run Locally
No install needed. Just:

1. Clone the repo:
```
git clone https://github.com/FarmanAli108/Habit-Tracker.git
```
2. Open `index.html` in your browser.
That's it. No build step, no dependencies.

## Features
- Add, rename, and delete habits
- Weekly grid with daily checkboxes
- Streak counter per habit
- Week navigation (prev/next/today)
- Data persists across page reloads via localStorage
- Responsive — works on mobile and desktop
